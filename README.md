# Nesrin Koç | Psikolog Kişisel Web Sitesi

Psikolog **Nesrin Koç** için geliştirilmiş modern, mobil uyumlu kişisel web sitesi.

---

## Proje Hakkında

Bu web sitesi; bir psikologun hizmetlerini, mesleki deneyimlerini ve iletişim bilgilerini profesyonel biçimde sunmak amacıyla tasarlanmıştır. Ziyaretçilerin ihtiyaç duydukları bilgiye hızla ulaşabilmesi için sade, temiz ve kullanıcı dostu bir arayüz benimsenmiştir.

### Sayfalar / Bölümler

| Bölüm | İçerik |
|---|---|
| **Hero** | Karşılama ve kısa tanıtım |
| **Hakkımda** | Psikolog hakkında genel bilgi |
| **Hizmetler** | Sunulan psikolojik danışmanlık hizmetleri |
| **Deneyim** | Mesleki geçmiş ve uzmanlık alanları |
| **İletişim** | Randevu ve ulaşım bilgileri |

---

## Teknolojiler

| Teknoloji | Sürüm | Kullanım Amacı |
|---|---|---|
| [Next.js](https://nextjs.org/) | 14 | React framework, App Router |
| [Tailwind CSS](https://tailwindcss.com/) | 3 | Utility-first CSS |
| [TypeScript](https://www.typescriptlang.org/) | 5 | Tip güvenliği |
| [Framer Motion](https://www.framer.com/motion/) | 12 | Sayfa animasyonları |
| [Lucide React](https://lucide.dev/) | — | İkon kütüphanesi |

---

## Özellikler

- Mobil öncelikli (mobile-first) duyarlı tasarım
- Hızlı sayfa yükleme — Next.js App Router ile optimize edilmiş
- Framer Motion ile akıcı geçiş animasyonları
- Erişilebilir ve sade kullanıcı arayüzü
- Bileşen tabanlı, sürdürülebilir mimari
